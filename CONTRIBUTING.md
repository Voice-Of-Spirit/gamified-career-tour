# Contributing to Gamified Career Tour

We welcome contributions to the Gamified Career Tour project! This document provides guidelines and instructions for contributing.

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/YOUR_USERNAME/gamified-career-tour.git`
3. Create a feature branch: `git checkout -b feature/your-feature-name`
4. Install dependencies: `npm install`

## Development Setup

1. Copy `.env.example` to `.env` and update with your local configuration
2. Start the backend: `npm run dev`
3. Start the frontend: `npm start` (from frontend directory)

## Commit Guidelines

- Use clear and descriptive commit messages
- Prefix commits with type: `feat:`, `fix:`, `docs:`, `test:`
- Example: `feat: add achievement system for career milestones`

## Pull Request Process

1. Update README.md with any new features
2. Ensure all tests pass
3. Request review from maintainers
4. Address any feedback

## Code Style

- Follow JavaScript/React conventions
- Use meaningful variable names
- Add comments for complex logic
- Keep functions small and focused

## Report Issues

Use the Issues tab to report bugs and suggest features with detailed information.

Thank you for contributing!
